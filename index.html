Antes de comenzar a estudiar jQuery, es importante tener un conocimiento básico de HTML, CSS y JavaScript. Estas son las bases fundamentales para comprender y utilizar eficazmente la biblioteca de jQuery.

  • HTML
  • CSS
  • JavaScript

Al tener un conocimiento básico de HTML, CSS y JavaScript, podrás aprovechar al máximo las capacidades de jQuery y utilizarlo de manera efectiva para simplificar y mejorar tus desarrollos web.

Si desea estudiar estos temas primero, busque los tutoriales en la pagina W3 Schools

jQuery es una biblioteca liviana de JavaScript que se enfoca en simplificar el uso de JavaScript en un sitio web, siguiendo el lema "escribe menos, haz más".

El propósito principal de jQuery es facilitar la implementación de tareas comunes en JavaScript, las cuales, de otra manera, requerirían varias líneas de código, y las encapsula en métodos que se pueden invocar con una sola línea.

Además, jQuery simplifica aspectos complicados de JavaScript, como las llamadas AJAX y la manipulación del DOM.

La biblioteca jQuery incluye las siguientes características:

  • Manipulación de HTML/DOM
  • Manipulación de CSS
  • Métodos de eventos HTML
  • Efectos y animaciones
  • AJAX
  • Utilidades

Nota: Adicionalmente, es importante destacar que jQuery cuenta con una amplia variedad de complementos disponibles para abordar prácticamente cualquier tarea.

Existen varias bibliotecas de JavaScript disponibles, sin embargo, jQuery se destaca como la opción más popular y versátil.

Un gran número de las principales empresas en Internet utilizan jQuery, algunas de las cuales incluyen:

  • Google
  • Microsoft
  • IBM
  • Netflix

Grandes empresas como Google, Microsoft, IBM y Netflix confían en jQuery para desarrollar sus aplicaciones y sitios web. Su amplia adopción demuestra su capacidad para ofrecer una experiencia de usuario excepcional y garantizar la compatibilidad en diferentes navegadores.

El equipo de desarrollo de jQuery ha priorizado la compatibilidad entre navegadores y se ha dedicado a garantizar que la biblioteca funcione de manera consistente en los principales navegadores utilizados actualmente, como:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

La compatibilidad de jQuery se extiende a una amplia variedad de navegadores, lo que significa que al utilizar jQuery en su sitio web, puede confiar en que ofrecerá una experiencia fluida y correcta para los visitantes, independientemente del navegador que utilicen.

La amplia aceptación y adopción de jQuery por parte de empresas y desarrolladores se debe, en parte, a su capacidad para proporcionar una compatibilidad confiable y uniforme en diferentes navegadores. Esto simplifica el desarrollo y garantiza una experiencia de usuario coherente en todas las plataformas.

Implementacion

Agregar jQuery a su página web

Descargar jQuery

Existen dos versiones de Jquery para su descarga:

  • Versión de producción: esto es para su sitio web en vivo porque ha sido minimizado y comprimido

  • Versión de desarrollo: esto es para pruebas y desarrollo (código sin comprimir y legible)

La biblioteca jQuery es un solo archivo JavaScript y se hace referencia a él con la etiqueta HTML script, esta etiqueta debe estar dentro de la seccion head:

   <head>
    <script src="jquery-3.6.4.min.js"></script>
    </head>

jQuery CDN

Si prefieres evitar la descarga y alojamiento de jQuery en tu servidor, puedes optar por incluirlo desde una CDN. Una CDN es una red de servidores que almacenan y entregan contenido de manera eficiente.

Al utilizar una CDN para jQuery, simplemente enlazas a la versión alojada en la red, sin necesidad de administrar el alojamiento y las actualizaciones en tu servidor. Esto simplifica la implementación y garantiza una carga rápida y confiable de jQuery en tu sitio web.

   <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    </head>

Sintaxis

La sintaxis de jQuery

Sintaxis basica

La sintaxis de jQuery se destaca por su eficiencia al seleccionar elementos HTML y ejecutar acciones sobre ellos, simplificando así la manipulación dinámica del contenido web.

       
              $( selector ). acción ()

  • Un signo $ para definir/acceder a jQuery

  • Un ( selector ) para "consultar (o encontrar)" elementos HTML

  • Una acción jQuery () que se realizará en los elementos

En resumen, La sintaxis de jQuery permite seleccionar y modificar elementos HTML de forma dinámica y eficiente.

   $(this).hide()

Esta expresión en jQuery se utiliza para ocultar el elemento HTML al que se está aplicando. $(this) se refiere al elemento actual en el contexto en el que se ejecuta el código, y .hide() es un método de jQuery que oculta ese elemento, haciendo que no sea visible en la página.

   $("p").hide()

Esta expresión en jQuery se utiliza para ocultar todos los elementos de párrafo (<p>) en la página. $("p") selecciona todos los elementos de párrafo presentes en el documento HTML, y .hide() es un método de jQuery que oculta esos elementos, haciendo que no sean visibles en la página.

   $(".test").hide()

Esta expresión en jQuery se utiliza para ocultar todos los elementos que tienen la clase CSS "test" en la página. $(".test") selecciona todos los elementos que tienen la clase "test" y .hide() es un método de jQuery que oculta esos elementos, haciendo que no sean visibles en la página.

   $("#test").hide()

Esta expresión en jQuery se utiliza para ocultar el elemento que tiene el identificador (ID) "test" en la página. $("#test") selecciona el elemento con el ID "test" y .hide() es un método de jQuery que oculta ese elemento, haciendo que no sea visible en la página.

El evento Document Ready

   $(document).ready(function(){

  // jQuery methods go here...

});

Para evitar que se ejecute el código jQuery antes de que el documento haya finalizado su carga y esté listo, es recomendable adoptar la práctica de esperar a que el documento esté completamente cargado.

Esto permite ubicar el código JavaScript antes del cuerpo del documento, en la sección del encabezado, y garantiza una manipulación adecuada del documento una vez que esté disponible.

Aquí hay algunos casos en los que las acciones pueden fallar si los métodos se ejecutan antes de que el documento se haya cargado por completo:

  • Intentando ocultar un elemento que aún no se ha creado

  • Intentando obtener el tamaño de una imagen que aún no está cargada

El equipo de jQuery ha desarrollado un método más breve para este evento:

   $(function(){

  // jQuery methods go here...

});

Utilice la sintaxis que prefiera. Creemos que el evento de documento listo es más fácil de entender al leer el código.

SELECTORES

una de las partes más importantes de la biblioteca jQuery


Los selectores de jQuery se utilizan para "encontrar" (o seleccionar) elementos HTML en función de su nombre, identificación, clases, tipos, atributos, valores de atributos y mucho más.

Todos los selectores en jQuery comienzan con el signo de dólar y paréntesis : $().

El selector de elementos

El selector de elementos jQuery selecciona elementos en función del nombre del elemento.

Un ejemplo seria el poder seleccionar todos los elementos <p> en una página

$("p")

El selector de #id

El selector jQuery usa el atributo id de una etiqueta HTML para encontrar el elemento específico .#id

Para encontrar un elemento con una identificación específica, se usa la siguiente sintaxis

$("#test")

El selector .class

El selector jQuery .classencuentra elementos con una clase específica

Para encontrar un elemento con una clase específica, se usa la siguiente sintaxis

$(".name_class")

EVENTOS

Un evento representa el momento preciso en que sucede algo.

Todas las diferentes acciones de los visitantes a las que una página web puede responder se denominan eventos.

El término "incendios/despedidos" se usa a menudo con eventos. Ejemplo: "El evento de pulsación de tecla se activa en el momento en que pulsa una tecla".

A continuacion se presenta una tabla con algunos eventos DOM comunes:
